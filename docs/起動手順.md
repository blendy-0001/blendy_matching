# 🤝 協業マッチングシステム 起動手順

## 初回セットアップ（最初の1回だけ）

### 1. Pythonのインストール確認
```
python --version
```
3.10以上であればOK。なければ https://python.org からインストール。

### 2. ライブラリのインストール
```
cd C:\Users\yo_yo\Documents\blendy_matching
pip install -r requirements.txt
```

### 3. 環境変数の設定
`.env.example` をコピーして `.env` を作成し、2つのAPIキーを入力：

```
NOTION_API_KEY=secret_xxxx   ← Notion Integration Tokenを貼る
CLAUDE_API_KEY=sk-ant-xxxx   ← Claude APIキーを貼る
```

**Notion Integration Tokenの取得方法：**
1. https://www.notion.so/my-integrations を開く
2. 「新しいインテグレーション」を作成
3. 「🤝 協業マッチングシステム」ページをそのインテグレーションと接続する
4. トークンをコピーして .env に貼る

---

## 毎回の起動方法

```
cd C:\Users\yo_yo\Documents\blendy_matching
python main.py
```

ブラウザで → http://localhost:8000 を開く

---

## 使い方（月2回の手順）

1. `python main.py` でサーバー起動
2. http://localhost:8000 を開く
3. 統計を確認（総登録者数・マッチング可能件数・累計）
4. 「▶ マッチング実行」をクリック
5. AIが分析中...（メンバー数により1〜5分）
6. 結果が表示される
7. 各ペアの紹介文を「コピーする」でコピー
8. LINE / Facebookでグループを作成して貼り付けて送信
9. Notionの「📋 マッチング結果」に自動保存済み

## フォルダ構成

```
blendy_matching/
├── main.py              ← Webアプリ本体
├── matching_engine.py   ← AIマッチングロジック
├── notion_client.py     ← Notion連携
├── config.py            ← 設定（DB IDなど）
├── requirements.txt     ← 必要ライブラリ
├── .env                 ← APIキー（自分で作成）
├── .env.example         ← .envのテンプレート
├── 起動手順.md           ← この手順書
└── templates/
    └── index.html       ← ダッシュボード画面
```
