Claude（Anthropic）APIキー 取得手順

本サービスのAI機能（会社プロフィール生成・商談議事録の分析）に使用します。所要 約10分。

このガイドは、Claude（Anthropic社のAI）のAPIキーを取得し、安全に共有いただくための手順書です。 専門知識は不要です。画面の案内に沿って進めてください。

はじめに大切なこと: APIキーの利用にはお支払い設定（クレジットの購入）が必要です。 アカウントを作っただけでは利用できません（手順 4 で設定します）。費用は使った分だけの従量課金で、 通常のご利用では月あたり数百円〜数千円程度の少額です。

手順

  1. アカウントを作成する

    https://console.anthropic.com を開き、メールアドレス（または Google アカウント）で サインアップします。届いた確認メールのリンクをクリックして認証してください。

  2. 組織情報を入力する

    初回ログイン時に組織名（会社名）などの簡単な情報を求められます。入力して進めます。

  3. 電話番号を認証する

    セキュリティのため、SMS による電話番号認証を求められる場合があります。届いたコードを入力します。

  4. お支払い設定・クレジット購入（重要）

    左メニューの 「Billing」（お支払い） を開き、クレジットカードを登録して クレジットを購入します（例: $5〜$20 程度で十分に始められます）。 これを行わないと API キーが機能しません。

    使い過ぎが心配な場合は、同じ Billing 画面の 「Usage limits」（利用上限）で 月額上限を設定できます（例: 月 $20 まで）。
  5. APIキーを発行する

    左メニューの 「API Keys」 を開き、「Create Key」ボタンを押します。 キーの名前（例: blendy）を入力して作成します。

    注意: 作成直後に表示される文字列（sk-ant- で始まる）は その場でしか表示されません。必ずコピーして控えてください（再表示はできません）。
  6. キーを安全に共有する

    コピーした APIキーを、開発担当へ安全な方法でお渡しください（次ページの注意を参照）。

APIキーの見た目（例）

正しく取得できると、次のような文字列になります（実際の値は伏せています）。

sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

共有時のセキュリティ注意

やること避けること
パスワード管理ツールや暗号化された手段で渡す SNS・公開チャットへの貼り付け
渡す相手（開発担当）を限定する メール本文に平文で記載して不特定多数に転送
不要になったらコンソールから無効化（Delete）できる スクリーンショットで SNS 共有
万一キーが漏れても、Anthropic コンソールの「API Keys」からいつでも無効化・再発行できます。ご安心ください。

送付前チェックリスト

  • ☐ アカウントを作成し、メール認証を完了した
  • ☐ Billing でクレジットを購入した（これが無いと動きません）
  • ☐ 「Create Key」でキーを発行し、sk-ant- の文字列を控えた
  • ☐ 安全な手段で開発担当に共有する準備ができた

よくある質問

Q. 無料で使えますか？
A. ごく少額の従量課金です。クレジット（前払い）を購入してご利用いただきます。使った分だけ消費されます。

Q. どれくらい費用がかかりますか？
A. 通常のご利用では月あたり数百円〜数千円程度が目安です。心配な場合は利用上限を設定できます。

Q. キーを間違えて公開してしまいました。
A. コンソールの「API Keys」で該当キーを Delete（無効化）し、新しいキーを作り直してください。
