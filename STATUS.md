## 📞 トラブルシューティング

### 問題: 「紹介文の生成に失敗しました」と表示
**原因**: Claude API がレート制限または認証エラー
**解決**: 新しい API キーで .env を更新

### 問題: メンバーが表示されない
**原因**: Notion API キーまたはデータベースアクセス権限の問題
**確認**: Notion データベースで "ブレンディマッチング" が共有されているか確認

### 問題: Notionに保存されない
**原因**: データベース ID が間違っている、またはアクセス権限なし
**確認**: config.py 内の MEMBERS_DB_ID, MATCHING_RESULTS_DB_ID, MATCHING_HISTORY_DB_ID
