## 🎓 トラブルシューティング

### Q: テスト後、「紹介文の生成に失敗しました」と表示される
**A**: Claude API キーがまだ有効でない可能性があります。以下を確認:
- API キーが正しく入力されているか
- キーが最新のものか（旧キーは 2026-06-01 まで無効）
- インターネット接続確認
- 数秒待ってから再実行

### Q: Notion に保存されない
**A**: 以下を確認:
- Notion API キーが有効か
- データベース ID が正しいか
- Notion で「ブレンディマッチング」が共有されているか

### Q: 本番環境が更新されない
**A**: Render.com の自動デプロイを確認:
1. GitHub リポジトリ確認
2. Render.com ダッシュボードで最新デプロイを確認
3. 数分待つ（デプロイに 2～3分かかります）
